# Mini Web Arcade

Mini-Web-Arcade provides a fun and entertaining way to play a few retro arcade games with simple interactive keyboard controls that respond onclick. A dynamic UX experience and design awaits every user attempting to try out and visit our up-to-date interactive website, decorated with fonts that are easy on the eyes. User logins are a minimum requirement to ensure security back-and-forth to our website when basic level user creation has been done.

Just chillax and have lots of fun!~
